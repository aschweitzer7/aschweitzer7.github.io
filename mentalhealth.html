Initial Mockup

Semester 1 Work

Core Client Deliverables: Product Vision, User Research Writeup, Miminum Viable Product and User Stories, Preliminary User Interface Mockups, Usability Evaluation, Initial Prototype.

Semester 1 of the project (Aug. 2021 - Dec. 2021) focused on gathering requirements and an ultimate client vision, as well as conducting user research and producing effective product mockups.

Semester 2 Work

Core Client Deliverables: Demo Videos for Coding Sprints, GitHub Source Code Delivery, Publication to the App Store and Google Play Store.

Semester 2 of the project (Jan. 2022 - May 2022) focused on project implementation such as programming the application, providing our client with an interactive database, and refining UI/UX designs and branding such as an app logo and testing for optimal user experience.

Technologies and Tools

Preliminary designs for the user interface of the mobile application are created using Figma, to be referenced during front-end app development while utilizing libraries. React Native is an open-source JavaScript framework that our team is using for developing our mobile application on both iOS and Android platforms. Information is stored in the database MongoDB.
